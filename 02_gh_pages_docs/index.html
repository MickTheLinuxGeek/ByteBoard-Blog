Byte Board Blog

A modern blogging platform powered by Django and ReactPy.

About the Project

Byte Board Blog is a full-featured blogging application built with Python and the Django web framework. It utilizes ReactPy to create dynamic and interactive user interfaces.

Features

  • Dynamic Frontend: Built with ReactPy for a modern, component-based frontend.
  • REST API: Includes a REST API for content management and integration with other services.
  • SEO Friendly: Comes with a sitemap generator for better search engine visibility.
  • Extensible: Built on Django, a powerful and flexible web framework.

Getting Started

To get a local copy up and running, follow these simple steps.

# 1. Clone the repo
git clone https://github.com/mickthelinuxgeek/byte-board-blog.git

# 2. Install dependencies
uv pip install -r requirements.txt

# 3. Run database migrations
python manage.py migrate

# 4. Start the development server
python manage.py runserver
